class BaseContract(name, ctor_params, wc=0, address=None, override_address=None, pubkey=None, private_key=None, balance=100000000000, nickname=None)[source]¶

The BaseContract object, which responsible for deploying contracts and interaction with deployed contracts.

__init__(name, ctor_params, wc=0, address=None, override_address=None, pubkey=None, private_key=None, balance=100000000000, nickname=None)[source]¶

Constructs BaseContract object.

Parameters

  • name – Name used to load contract’s bytecode and ABI

  • ctor_params – Parameters for offchain constructor call If None constructor is not called and can be called with separate call_method() call (onchain constructed)

  • wc – workchain_id to deploy contract to

  • address – If this parameter is specified no new contract is created but instead a wrapper for an existing contract is created

  • override_address – When specified this address will be used for deploying the contract otherwise the address is generated according real blockchain rules

  • pubkey – public key used in contract construction

  • private_key – private key used to sign construction message

  • balance – desired contract balance

  • nickname – nickname of a contract used in verbose output

call_getter(method, params={}, key=None, decode_ints=True, decode_tuples=True, dont_decode_fields=[], expect_ec=0, decode=False)[source]¶

Calls a given getter and decodes an answer.

Parameters

  • method – Name of a getter

  • params – A dictionary with getter parameters

  • key – (Optional) If function returns tuple this parameters forces to return only one value under the desired key.

  • dont_decode_fields – A list of fields in answer that should not be decoded

  • expect_ec – Expected exit code. Use non-zero value if you expect a getter to raise an exception

Returns

A returned value in decoded form (exact type depends on the type of getter)

Return type

type
